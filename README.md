# GymSmart ERP Management

Welcome to **GymSmart**, a premium gym management and booking platform built with Next.js 16 (Turbopack) and Tailwind CSS.

## Features
- **Public Landing Page**: Premium marketing page with built in scheduling, online booking, BMI calculator, and facility showcase.
- **ERP Dashboard**: A secure backend management portal for tracking sales, managing members, trainers, library, store, and more.

## Tech Stack
- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- ApexCharts
- Lucide React Icons

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
